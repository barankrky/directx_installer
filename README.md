# DirectX Runtime Installer for Windows

- Completely silent automatic installation.
- Tested on Windows 10

## Silent Installation 
- Silent version. Installation becomes completely silent and progress will show up in the notification area.

![image](https://github.com/barankrky/directx_installer/assets/53481486/300f3b45-00b5-4605-bc05-f713df1fc5cf)

## Console Installation
- Console version. Installing progress will show up in a command prompt.
